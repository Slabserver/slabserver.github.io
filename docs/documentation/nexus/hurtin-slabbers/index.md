# Hurtin' Slabbers

Hurtin' Slabbers by maketues. It started as a survival build on Slab. The Nexus version keeps the game identical but adds instancing, protected blocks, and a proper save system so progress carries between sessions.

![hurtin slabbers](/assets/images/nexus/hurtin-slabbers.png)

There is a leaderboard to track your time and compete with other slabbers.

The game has 8 levels and a boss fight at the end. Expect around 90 minutes to complete it from start to finish, depending on your skill level. You do not need to do it in one sitting.

# How to Play

## The Game

Hurtin' Slabbers is a linear platformer. Each level is a series of obstacles that test your Minecraft movement skills. Most levels have more than one route through.

There are 8 levels in total, ending in a boss fight. Completing the whole game takes around 90 minutes depending on skill. You do not need to finish it in one go.

## Checkpoints

Checkpoints sit between levels. When you reach one, click then sign THEN stand on the campfire to die. You should see the message to confirm it worked.

If you die, you respawn at your last checkpoint.

## Saving and Quitting

You do not need to finish in one session. The server will return you.
