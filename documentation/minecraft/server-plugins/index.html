    Server Plugins¶

    Info

    * Plugin is written and maintained by the staff team

    ** Plugin is deprecated and subject to future removal

    Infrastructure & Gameplay¶

    Plugin Description
    {Survival} AFKSleep* Handles multiplayer sleep, AFK/Idle states and Discord !playerlist
    {Survival} BlockPlaceLog* Tracks Easyplace usage in Litematica
    {Survival} Bouncer* Handles Resource World infrastructure (Bungeecord level) and network whitelist
    {Survival} Buster* Enables dynamic scaling of view and simulation distance, player counts, and mob caps
    {Both} DiscordSRV Handles ingame chat synchronisation to Discord for the Survival and Creative servers
    {Survival} DirectoryPlugin* Handles stock summaries in the Google Sheet for the directory
    {Survival} DragonCandle* Explore... Descend... Discover...
    {Creative} F3NPerm Enables gamemode switching using F3 + N and F3 + F4 on the Creative server
    {Survival} FastChunkPregen Pre-generates chunks, primarily in the Resource World
    {Survival} Fireworks* Adds firework recipes to the crafting recipe book
    {Survival} Gatekeeper* Handles Resource World infrastructure (Paper level)
    {Survival} HuskSync Synchronises player data to and from the Resource World
    {Survival} LightningFire* ** Prevents lightning from starting fires and torching bases
    {Survival} Pl3xMap Adds map.slabserver.org
    {Survival} RawStats* Displays various player statistics in the sidebar, when enabled by staff
    {Both} Shrug* Adds the /shrug command to easily send the ¯\_(ツ)_/¯ emoticon in chat
    {Survival} ShulkerSaver* ** Prevents Shulker boxes being deleted by MC-902, prior to the bugfix in Minecraft 1.21
    {Survival} SeedWriter* Logs each Resource World seed for future reference
    {Survival} TAB Provides custom tablist configuration
    {Survival} TAB - Bridge Provides custom tablist configuration (Bungeecord level)
    {Survival} VentureChat Improves chat features, and syncs chat across the Survival servers via Bungeecord
    {Both} Wrench* Provides easier rotation of a wide varity of blocks

    Administration¶

    Plugin Description
    AdvancedBan Allows for synchronized bans across the network and tempbans
    Camera* A better, survival-admod-friendly spectator mode
    Chunky* A quick and efficient chunk pre-generator
    CollarCheck* Pet owner checks
    CoreProtect Logs interactions in the world to a database
    EntityDetection Helps with lagbusting by finding chunks with high amounts of entities
    EntityGlo* Helps with lagbusting by highlighting entities in a loaded area
    InventoryRollbackPlus Helps to easily restore player inventories if you die to glitches or bugs
    LessEndermanGriefing Reduces the blocks that Enderman can pick up, based on their current dimension
    LuckPerms Handles permission groups of staff and players
    OpenInv Allows remote access to player inventories and Ender Chests
    PremiumVanish Allows us to hide our presence on the server and observe xraying attempts
    Plan Provides player analytics for the activity and geological distribution of our playerbase
    Spark Server health monitoring and the /tps command
    WorldEdit Integrates with CoreProtect for easier operations
