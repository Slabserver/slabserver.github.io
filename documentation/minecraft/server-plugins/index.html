    Infrastructure & Gameplay

    Plugin Description
    AFKSleep Handles multiplayer sleep, AFK/Idle states and !playerlist command in Discord
    AxiomPaperPlugin Adds serverside support for Axiom
    BlockPlaceLog Tracks usage of Easyplace feature in Litematica
    Bouncer Handles Resource World infrastructure (Bungeecord level) and network whitelist
    BungeePlayerList Enables the on-hover playerlist in the server select menu for the Bungeecord network
    Buster Adds dynamic scaling of view and simulation distance, player counts, and mob caps
    Chunky Pre-generates chunks, primarily used for the Resource World resets
    DiscordSRV Synchronises Minecraft's ingame chat to and from our Discord channels
    DragonCandle Explore... Descend... Discover...
    F3NPerm Enables gamemode switching via F3 + N and F3 + F4 on the Creative server
    Fireworks Adds firework recipes to the crafting recipe book
    Gatekeeper Handles Resource World infrastructure (Paper level)
    HuskSync Synchronises player data to and from the Resource World
    InvisItemFrames Adds a feature to turn Item Frames invisible with a Splash Potion of Invisibility
    LightningFire Prevents lightning from starting fires and torching bases
    Pl3xMap Adds map.slabserver.org
    Pl3xMapExtras Adds ability to show mobs/signs/banners on Pl3xMap
    RawStats Displays various player statistics in the sidebar (when enabled by staff)
    Shrug Adds the /shrug command to easily send the Â¯\_(ãƒ„)_/Â¯ emoticon in chat
    ShulkerSaver Prevents Shulker boxes being deleted by MC-902, prior to the bugfix in Minecraft 1.21
    SimpleBack Adds a /back command to return to your previous location
    SimpleReply Adds a /r command to reply to private messages
    TAB Provides custom tablist configuration
    TAB - Bridge Provides custom tablist configuration (Bungeecord level)
    VentureChat Adds various chat features, and syncs chat across the Survival servers via Bungeecord
    Wrench Provides easier rotation of a wide varity of blocks

    Administration

    Plugin Description
    AdvancedBan Allows for synchronized bans across the network and tempbans
    Bouncer-Lite Handles network whitelist for our servers not within the Bungeecord network
    Camera Adds an improved survival-friendly spectator mode for staff
    CollarCheck Allows staff to check the owner of a tameable animal
    CoreProtect Logs interactions in the world to a database
    DirectoryPlugin Handles stock summaries in the Google Sheet for the directory
    EntityDetection Helps with lagbusting by finding chunks with high amounts of entities
    EntityGlo Helps with lagbusting by highlighting entities in a loaded area
    InventoryRollbackPlus Helps to easily restore player inventories if you die to glitches or bugs
    LessEndermanGriefing Reduces blocks that Enderman can pick up, based on their current dimension
    Lite2Edit Converts Litematics to WorldEdit schematics
    LuckPerms Handles permission groups of staff and players
    OpenInv Allows remote access to player inventories and Ender Chests
    PremiumVanish Allows us to hide our presence on the server and observe xraying attempts
    Plan Provides player analytics for the activity and geological distribution of our playerbase
    PlotSquared Provides plot areas for players to claim and manage
    PVPToggle Helps staff easily disable PVP during major community events
    SeedWriter Logs each Resource World seed for future reference
    Spark Adds server health monitoring and the /tps command
    WorldEdit Adds map editor features on Creative, and integrates with CoreProtect
    WorldGuard Adds several features to protect the Creative spawn from WorldEdit changes
    WorldGuard Extra Flags Adds more granular permission settings for WorldGuard
